# Aprendizaje Automático
## Proyecto final

En este git está subido el proyecto final de [Aprendizaje Automático](http://www.fdi.ucm.es/Pub/ImpresoFichaDocente.aspx?Id=722).
El proyecto se basa en clasificar páginas webs en diversos temas, a partir de su contenido. El dataset utilizado fue http://mldata.org/repository/data/viewslug/dmoz-web-directory-topics/
Para más información, en la memoria del trabajo se presenta todo más desarrollado.
